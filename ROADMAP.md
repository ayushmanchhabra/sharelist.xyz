# Roadmap

## Analysis

- Landing page for SEO.
- User logs in using Auth0.
- User is redirected to dashboard.
- User connects their streaming service.
- User is able to create music playlists.
- User shares the playlist with a friend.
- Friend logs in with their streaming service.
- Friend plays the playlist directly in app.
- If a song is not available in their streaming service then query other free services.

## Development

- React for website
- MUI for component development
- Tailwind for styling
- Go Gin server to handle users and payments
- PostgreSQL to store user and their playlist info
- Docker for local development
- Spotify API for searching for Spotify music
- Apple API for searching for Apple music
- YouTube API for searching for YouTube music
- Soundcloud API for searching for YouTube music
- Stripe API for collecting payments

- Vitest for unit testing
